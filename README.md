# Pre-Requisites

1. Install the latest node version **v22.7.0**

## Client Dependencies
1. Navigate to the client directory

``cd /client-sq``

2. Install client dependencies
``npm i``


## Server Dependencies
1. Navigate to the server directory

``cd /client-sq``

2. Install server dependencies
``npm i``
